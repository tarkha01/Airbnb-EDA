# Airbnb New User Booking – Exploratory Data Analysis (EDA)

This project performs **Exploratory Data Analysis (EDA)** on the Airbnb New User Booking dataset to understand patterns in user behavior and booking destinations.

The analysis focuses on exploring user demographics, registration trends, and potential relationships between features such as **age, gender, account creation date, and destination country**.

## Project Objectives

* Explore and understand the structure of the dataset
* Identify missing values and perform basic data cleaning
* Analyze user demographics such as age and gender
* Study trends in user registrations over time
* Investigate relationships between user features and booking destinations
* Create visualizations to better understand patterns in the data

## Dataset

The dataset comes from the **Airbnb New User Bookings Kaggle competition** and includes information about user accounts such as:

* User demographics (age, gender, language)
* Account creation and activity timestamps
* Device and browser information
* Destination country (for training data)

## Key Steps

* Data loading and preprocessing
* Handling missing values and unrealistic age values
* Feature engineering (date features and age groups)
* Exploratory data analysis using visualizations
* Analysis of user behavior across different demographics

## Tools and Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* XGBoost (imported for potential modeling)

## Results

The analysis reveals trends in:

* User age distribution
* Gender participation
* Registration growth over time
* Destination preferences across demographics

These insights can help guide further **machine learning models** for predicting user booking destinations.
